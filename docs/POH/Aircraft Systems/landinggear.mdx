title: Landing Gear
---
The landing gear is a tricycle configuration with steel landing gear legs. The nose wheel is free castering. The nose wheel tire is size 5.00-5 and is 6-ply. The main gear tires are size 15x6.00 x 6 and are 6-ply. PSI for main tires is: 42psi for nose tires: 40psi.
